# Игра в слова

Программа для поиска цепочки слов по правилам игры в слова.
Задача сводится к поиску Эйлерова цикла в ориентированном графе.


## Правила игры

- Очередное слово должно начинаться на букву, которой закончилось предыдущее слово
- Если слово заканчивается на «ь», используется предыдущая буква
- Последнее слово цепочки должно заканчиваться на первую букву первого слова
- Достаточно найти одно подходящее решение
- Возможен вариант, что решения не существует


## Алгоритм

1. Построение ориентированного графа:
   - Вершины — буквы русского алфавита
   - Рёбра — слова (из первой буквы в эффективную последнюю)

2. Проверка существования Эйлерова цикла:
   - in-degree == out-degree для всех вершин
   - Граф связен в неориентированном смысле

3. Поиск Эйлерова цикла:
   - Алгоритм Иерхольцера (итеративная реализация)
   - Перебор всех возможных стартовых вершин

4. Формирование цепочки в односвязном списке


## Запуск

python main.py

  0. Выход


## Пример работы

Вход:
хмель мороз налог лимон запах дым лошадь холм гол молох

Выход:
хмель лимон налог гол лошадь дым мороз запах холм молох

Проверка:
хмель -> лимон -> налог -> гол -> лошадь -> дым -> мороз -> запах -> холм -> молох
